# SCT_ML_4

## Task 4 - Hand Gesture Recognition

This project implements a Hand Gesture Recognition model using Machine Learning.

### Gestures Used
- Fist
- Palm
- Thumbs Up

### Technologies Used
- Python
- NumPy
- Pillow (PIL)
- Scikit-Learn

### Dataset
The dataset contains images of:
- Fist gestures
- Palm gestures
- Thumbs Up gestures

### Algorithm
Support Vector Machine (SVM)

### Output
The model classifies hand gestures and predicts whether the image belongs to:
- Fist
- Palm
- Thumbs Up

### Accuracy
Current model accuracy: 33.33%
